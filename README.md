# Buckeyes for Harm Reduction Website

A modern, responsive website for Buckeyes for Harm Reduction (BHR), an Ohio State University student organization dedicated to harm reduction advocacy and community support.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with intuitive navigation
- **Accessible**: Built with accessibility in mind
- **Easy to Maintain**: Simple HTML/CSS/JavaScript structure
- **SEO Friendly**: Proper semantic HTML and meta tags

## Pages

- **Home** - Hero section with mission statement and key initiatives
- **About Us** - Organization story and detailed mission
- **What is Harm Reduction** - Educational content about harm reduction principles and strategies
- **Resources** - Comprehensive list of overdose prevention and treatment resources
- **Contact** - Contact form and information

## Getting Started

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser to view the site locally
3. Modify files as needed

### Deploying to GitHub Pages

1. Create a GitHub repository called `bhr_website` (or similar)
2. Push this folder to the `main` branch
3. In GitHub repository settings, enable GitHub Pages for the `main` branch
4. Your site will be available at `https://<your-username>.github.io/bhr_website/`

## Customization

### Colors

Edit the CSS variables in `assets/css/style.css`:

```css
:root {
    --primary-color: #CC0000;      /* Buckeyes Red */
    --secondary-color: #00264d;    /* Dark Blue */
    --accent-color: #f39c12;       /* Gold */
    /* ... other variables ... */
}
```

### Content

Simply edit the HTML files to update text, links, and information. The structure is straightforward and easy to follow.

### Contact Information

Update the contact information in `pages/contact.html` with your actual email, phone, and location.
